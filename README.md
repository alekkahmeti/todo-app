# Aplikacion për Listën e Detyrave

Ky projekt është një aplikacion i thjeshtë për Listën e Detyrave i bazuar në internet, i zhvilluar për të demonstruar përdorimin e Git dhe GitHub për menaxhimin e projekteve softuerike.

## Qëllimet e Projektit
- Implementimi i kontrollit të versioneve duke përdorur Git
- Publikimi i projektit në GitHub
- Krijimi i dokumentacionit profesional të projektit

## Karakteristikat
- Shtimi i detyrave të reja
- Shënimi i detyrave si të përfunduara
- Fshirja e detyrave

## Gjuhet e perdorura
- HTML
- CSS
- JavaScript

## Si të Drejtoni Projektin
1. Klononi depozitën nga GitHub
2. Hapni skedarin index.html në një shfletues interneti
